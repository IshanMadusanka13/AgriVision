# AI-Driven Ecosystem for Scotch Bonnet Cultivation

A mobile based AI ecosystem designed to assist smallholder farmers in Sri Lanka with Scotch Bonnet pepper cultivation. The system addresses disease detection, growth monitoring, quality grading, and adaptive planting layout.

## Features

- **Early-Stage Disease Detection**: Identifies diseases from smartphone photos in under 4 seconds.
- **Growth Monitoring & Fertilizer Advisory**: Provides growth stage detection and soil-specific fertilizer recommendations.
- **Automated Quality Grading**: Classifies peppers into Grades Aâ€“D using image analysis.
- **Adaptive Planting Layout**: Generates field-specific planting layouts based on soil, climate, and land shape.

## Tech Stack

- **Frontend**: React Native
- **Backend**: FastAPI
- **AI Models**: YOLOv8
